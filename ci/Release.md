## Wishlist Hub Playnite v1.1.0

### Destaques
- Sync automática ao adicionar jogos (debounce + fila)
- Tags `[WishlistHub] Synced` / Miss / Ignored / Error
- Settings UI (token, URL, bibliotecas)
- Menu Extensions + menu do jogo

### Instalação
1. Baixe `WishlistHub_Playnite_1_1_0.pext` (anexar após build no Windows com `ci/pack.bat`)
2. Playnite → Add-ons → Install from file
3. Token em https://wishlist-hub.paz.poa.br/ferramentas#playnite

### Addon browser
Manifestos em `ci/`. Ver `ci/ADDON_BROWSER.md` para abrir o PR em PlayniteAddonDatabase.
